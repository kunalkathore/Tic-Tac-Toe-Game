# Tic Tac Toe Game

A polished Tic-Tac-Toe game built as a single HTML file with integrated CSS and JavaScript.

## Features

- 2-player mode
- Player vs AI mode with a simple minimax-based AI
- Score tracking for X, O, and draws
- Animated board and symbol transitions
- Stylish glassmorphism UI with animated background effects
- Keyboard shortcut: `ESC` to restart the game

## How to Use

1. Open `main.html` in your browser.
2. Choose a game mode:
   - `2 Players` for local multiplayer
   - `vs AI` to play against the computer
3. Click an empty cell to place your mark.
4. Use the `New Game` button to restart the current round.
5. Use the `Reset Scores` button to clear all scores and start fresh.

## Notes

- The AI plays as `O` in versus-computer mode.
- The game board and logic are fully implemented in `main.html`.
